==================================
nmBlog plugin for whatsnew module
==================================

「はっぴぃ・りなっくす」で配布されている新着情報モジュール用の
nmBlog プラグインです。

nmblog フォルダを whatsnew モジュールフォルダ配下の plugins フォ
ルダにコピーしてください。

その後 whatsnew モジュールの管理画面から、設定が出来るようにな
りますので、他のモジュール同様に設定してご利用下さい。
